Project Index

This project uses a high-level US population study to investigate capabilities of Shiny and R Studio Presenter to create high-quality and interactive data products and presentations.

  • Slide presentation describing the Shiny application.
  • Shiny application running dynamic population study maps. The application may take a couple minutes to startup on the first visit.
  • Project README file with version information required to reproduce this study.
  • Project master branch which contains the code for the Shiny application (i.e., ui.R and server.R).
  • Project gh-pages branch which contains the static (and compiled) project pages.
